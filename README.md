# RC_Car

Building an arduino powered RC car. Mostly just trying to keep track of files for myself but feel free to use any of this to build your own RC car


### Hardware
#### Arduino Uno (2)
#### DRV8871 H-Bridge Controller (1)
#### NRF24L01 Radio (2)
#### Joystick (1)
#### 12V 200rpm DC motor (1)
#### 180 deg Servo (2)
